1. Выполнить

apt-get update
apt-get install python3-pip
sudo pip3 install pyubx2


2. В файле gnssConfig.json изменить путь до хранения дампов "dump_root"
разрешено использовать только абсолютный путь

3. убедиться что гнсс-модем доступен 

ping 192.168.1.55

4. запустить сбор данных, для этого из папки с коннектором выполнить

python3 main.py

Если сбор данных идет то в консоли будет отображаться количество сделанных записей.
